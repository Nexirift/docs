# Nexicontext

Nexicontext is a feature that allows accounts to add additional information to posts that may be missing from the post itself.

## Rules

<BadgeWrapper selected="Specific to Nexirift" />

Accounts must comply with the following rules on the official Nexirift instance:

- The provided context must be relevant to the original post.
- The provided context must cite highly credible sources.
- The provided context must not be offensive or inappropriate.
- The provided context must not be spam or promote scams.
